# crypto-wallet-connect

a wordpress ERC20 cryptocurrency wallet which uses metamask to connect and create user account and can also be used to login in to wordpress. The wallet needs to have permission administration panel in the admin section of wordpress to blacklist or allow users to sigin using the meta

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/crypto-wallet-connect.git
cd crypto-wallet-connect
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
